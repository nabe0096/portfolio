## サイト情報

| 項目 | 内容 |
|---|---|
| 公開URL | https://nabenokitchen.com |
| GitHubリポジトリ | https://github.com/nabe0096/portfolio |
| ホスティング | GitHub Pages |

---

## デプロイ方法

ファイルを編集したら、以下のコマンドを実行するだけ：

```bash
bash deploy.sh
```

これだけでGitHubに自動でアップロードされ、サイトに反映される。
反映まで1〜2分かかる場合がある。
